# Nexora Aggregation Use Cases

Aggregation is used for:

* Reputation calculation
* Growth analytics
* Team performance
* Project insights

---

# Reputation Aggregation

Calculate impact from:

* Missions completed
* Reviews received
* Contributions

Always keep calculation sources traceable.

---

# Analytics Examples

Developer:

* Completed missions
* Growth timeline
* Skill progress

Team:

* Delivery speed
* Collaboration activity

Project:

* Progress
* Contribution distribution

---

# Aggregation Rules

Prefer:

* Efficient pipelines
* Filtering early
* Limiting returned data

Avoid:

* Heavy calculations on every request

---

# Performance

For expensive analytics:

Consider:

* Cached results
* Background jobs
* Precomputed values

---

# Final Rule

Aggregation should create insights, not replace good data modeling.
